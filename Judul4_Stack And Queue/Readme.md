# **Judul: Program Riwayat Aktivitas**

## **1. Tujuan Program**

Program ini bertujuan untuk mengimplementasikan struktur data stack menggunakan array (list) dalam bahasa Python untuk menyimpan dan mengelola riwayat aktivitas pengguna dalam sebuah aplikasi. Melalui program ini, pengguna dapat menambahkan aktivitas baru ke dalam sistem, menghapus aktivitas terakhir yang dilakukan, serta menampilkan seluruh riwayat aktivitas yang tersimpan.
