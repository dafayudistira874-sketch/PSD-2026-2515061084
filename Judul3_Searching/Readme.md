# **Judul: Membuat Program pencarian produk di toko**
## **1. Tujuan Program**
Program ini dibuat untuk mencari produk yang ada di dalam daftar produk toko menggunakan Sequential Search. Program memeriksa setiap data produk secara berurutan dari awal hingga akhir sampai produk yang dicari ditemukan. Jika produk ditemukan, program akan menampilkan posisi produk berdasarkan rak/indeks yang ditempati produk yang dimulai dari rak/indeks 1, sedangkan jika tidak ditemukan maka program akan menampilkan pesan bahwa produk tidak tersedia di toko.
